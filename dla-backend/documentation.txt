USUÁRIOS
  Ação: Autentica usuário
  Rota: /auth
  Method: POST
  Parametros: email e password

  Ação: Pegar todos os usuários
  Rota: /getusers
  Method: GET
  Parametros: Nenhum

  Ação: Pegar usuário específico
  Rota: /getuser/1
  Method: GET
  Parametros: Apenas id como mostrado ai em ROTA

  Ação: Inserir novo usuário
  Rota: /insertuser
  Method: POST
  Parametros: nome, email, password

  Ação: Editar um usuário
  Rota: /updateuser
  Method: PUT
  Parametros: id, nome, email, password (Só envia o que for editar, apenas o id é obrigatório)
    
  Ação: Deletar um usuário
  Rota: /deleteuser/1
  Method: DELETE
  Parametros: id passado na url como mostrado ai em ROTA
